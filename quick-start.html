Quick Start Guide

  1. Open the Converter
    Click the "Launch Converter" button above to open the main application.
  2. Upload Your Markdown File
    Drag and drop your .md file onto the upload area, or click "Browse Files" to select it.
  3. Choose a Template
    Select from 6 professional templates based on your document type (ISO, Project, Homework, Letter, etc.).
  4. Customize Settings
    Adjust image size, enable/disable Table of Contents, and configure document-specific options.
  5. Convert & Preview
    Click "Convert & Preview" to see your document rendered in real-time.
  6. Export Your Document
    Download as HTML, generate a PDF, or print directly from your browser.

Markdown Syntax

YAML Frontmatter

Add metadata at the beginning of your file:

---
title: My Document
author: Your Name
version: 1.0.0
created: 2025-12-09
description: Document description
---

Supported Markdown Features

  • Headers (H1-H6)
  • Bold, italic, strikethrough
  • Lists (ordered, unordered, task lists)
  • Code blocks with syntax highlighting
  • Tables
  • Blockquotes
  • Images
  • Links (external and internal)
  • Horizontal rules
  • Mermaid diagrams
Tip: Download the sample markdown file to see examples of all supported features and proper YAML frontmatter formatting.

System Requirements

  • Modern web browser (Chrome, Firefox, Safari, Edge)
  • JavaScript enabled
  • Internet connection (for loading CDN libraries)
  • No installation or server required
