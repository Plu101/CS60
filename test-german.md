# Einführung

Dieser Text testet die Unterstützung für deutsche Umlaute.

## Umlaute testen

Die Qualität des Produkts überzeugt die Käufer.

### Kleinbuchstaben

- ä (a-Umlaut)
- ö (o-Umlaut)  
- ü (u-Umlaut)
- ß (Eszett/scharfes S)

### Großbuchstaben

- Ä (A-Umlaut)
- Ö (O-Umlaut)
- Ü (U-Umlaut)

## Beispieltext

Die schöne Müllerin trägt grüne Röcke. Äpfel und Übungen für große Änderungen.

## Code mit Umlauten

```python
# Beispiel mit Umlauten
def grüße():
    """Grüßt den Benutzer"""
    print("Schön, dass Sie da sind!")
    print("Größe: über 180cm")
    
grüße()
```

## Mathematik mit deutschem Text

Die Fläche $A$ beträgt: $A = l \times b$

Für größere Berechnungen:

$$
\text{Gesamtfläche} = \sum_{i=1}^{n} A_i
$$

## Schlussfolgerung

Alle Umlaute (ä, ö, ü, Ä, Ö, Ü, ß) sollten korrekt dargestellt werden.
